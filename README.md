# HF-API
## Usage
`python hf_pb_notifier.py [-h] [-r REQUESTSPERHOUR] [-k HACKFORUMSKEY]
                         [-p PUSHBULLETKEY]`

### optional arguments:
  `-r REQUESTSPERHOUR`
                        Sets the maximum requests per hour
                        
  `-k HACKFORUMSKEY`
                        Pass your Hackforums API key
                        
`  -p PUSHBULLETKEY`
                        Pass your Pushbullet API key
